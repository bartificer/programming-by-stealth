PBS 185 — Worked Example

Welcome to PBS 185 Worked Example! This sample HTML page is designed to illustrate how SASS can be used to generate CSS for styling a web page.

The instalment show notes walk readers through editing index.scss to generate the page's style sheet as index.css. To see the effect of your changes, first, compile the updated index.scss file to index.css with the command below, then refresh this oage in your browser:

npx sass index.scss index.css
