Since we last visited Cascading Style Sheets back in instalment TO DO the language has continued to steadily improve. For the most part these changes have been incremental, but there is one change that's a lot more revolutionary than evolutionary — the introduction of a variable-like feature officially known as *custom properties*, but usually referred to as simply *CSS variables*. Custom properties tackle the same problem traditional variables do, but they do so in a very CSS way.
