## Laying the Foundations

In order to customise Bootstrap via SASS,  you need Bootstrap's source code rather than the compiled CSS. You can download this manually from their website or from GitHub, but you can also get it from common package repositories like NPM.

To keep things simple in our examples, we'll fetch our copy of Bootstrap from NPM.

To play along with the worked examples, create a new folder on your computer, then open a terminal in that new folder. In that terminal window, install Bootstrap into the folder with the command:

```sh
npm install bootstrap
```

For our purposes we now have two important resources:

1. A copy of the Bootstrap JavaScript in `./node_modules/bootstrap/dist/js/bootstrap.bundle.min.js`
2. A copy of the Bootstrap SASS code in the folder `./node_modules/bootstrap/sass`

To customise bootstrap we need to create our own SASS file that imports the Bootstrap SASS code.

Depending on the level of customisation you need, you can do this in two ways.

### Skeleton 1 — Basic Customisations:

If you don't need to alter Bootstrap's maps, and you don't need to make use of any Bootstrap functions when overriding the Bootstrap variables, then you can adopt this simple approach:

```scss
// Step 1 — override Bootstrap variables here
// CAVEATS:
// 1. you can't customise Boostrap's maps with this approach
// 2. you can't use the functions Bootstrap provides while overriding the variables
// …

// Step 2 — import Boostrap all at once
@import "../node_modules/bootstrap/scss/bootstrap";

// Step 3 — add your own SASS code, which can make use of:
// 1. Bootstrap's variables (for reading only)
// 2. Bootstrap's maps
// 3. Bootstrap's functions
// 4. Bootstrap's mixins
// 5. Boostrap's classes
// …
```

### Skeleton 2 — Deep Customisation

If you do want to customise Bootstrap's maps, or, if you need to use Bootstrap's functions when overriding Bootstrap's variables, then you need to adopt this more complex approach:

```scss
// Step 1 — import just the Bootstrap functions
@import "./node_modules/bootstrap/scss/functions";

// Step 2 — override Bootstrap variables here
// …

// Step 3 — import Bootstrap's variables
@import "./node_modules/bootstrap/scss/variables";
@import "./node_modules/bootstrap/scss/variables-dark";

// Step 4 — edit Bootstrap maps here
// …

// Step 5 — Include the rest of Bootstrap
@import "./node_modules/bootstrap/scss/bootstrap

// Step 6 — 
```

### Full Customisation

Note that if you need even deeper customisation, like using the Bootstrap API, then you need to do an extremely laborious extended import where you import exactly the Bootstrap components you want one-by-one, which can result in tens of explicit imports. You can [learn how to do this in Bootstrap's documentation](https://getbootstrap.com/docs/5.3/customize/sass/#importing), but it's beyond the scope of this series.
